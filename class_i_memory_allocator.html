template<class Type>
virtual IPointers<Type>& IMemoryAllocator< Type >::alloc ( size_t  _size)
pure virtual

It allocates the first available memory unit (number of elements of PointerType with SmartAllocator and number of bytes with PoolAllocator). If size equals to zero and then container will contain zero pointer.

Exceptions
std::bad_alloc()
Parameters
[in]size- the size of the requested memory in bytes.
Returns
IPointers<PointerType>& - a reference to pointer container
Warning
This method may allocate more memory than one needs. It's always multiple of 8 bytes and never less than that.

Implemented in PoolAllocator< PointerType, MemoryType >, and SmartAllocator< PointerType, MemoryType >.

virtual void IMemoryAllocator< Type >::realloc ( IPointers< Type > &  pointer,
size_t  _size 
)
pure virtual

It reallocates the first available units in the pool with new size. The memory is not initialized. If pointer equals zero behaves like alloc(_size). If size == 0 && pointer != 0 behaves like free(pointer).

Parameters
[in]pointer- reference to container with pointer to memory to be reallocated
[in]size- the size of the requested memory
